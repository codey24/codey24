##  Projects

### Development of Web-Based Monitoring System for Tablet Feeding System
**Mar 2024 - Jun 2024**

*Associated with Universiti Kuala Lumpur*

The Development of Web-Based Monitoring System for Tablet Feeding System project involved creating an integrated system to enhance the efficiency and reliability of a tablet feeding machine. As the lead programmer, I utilized Omron CJ2M PLCs and Node-RED for seamless data acquisition, processing, and real-time monitoring. This project included developing a web-based dashboard for remote monitoring, implementing predictive maintenance using cycle time data, and integrating various communication protocols such as FINS and MQTT.

The system was designed to collect and analyze data, detect anomalies using AI-based anomaly detection algorithms, and provide maintenance alerts to minimize downtime and optimize machine performance. Programming languages used in this project included Python for data analysis and anomaly detection, and JavaScript for developing the Node-RED flows and web-based dashboard.

**Skills**: Electricity 路 PLC Programming 路 Node-RED 路 Data Acquisition 路 Industrial Automation 路 Artificial Intelligence (AI) 路 JavaScript 路 Real-time Data Acquisition 路 Programmable Logic Controller (PLC) 路 Data Analysis 路 Predictive Maintenance 路 Python (Programming Language) 路 Anomaly Detection

### Development of Mini Conveyor System with Mobile Device Control and Monitoring

*Associated with Universiti Kuala Lumpur*

The Development of Mini Conveyor System with Mobile Device Control and Monitoring project was a freelance PLC programming task for a university student's final year project. ### Teras Jernang Wastewater Treatment Plant Automation Simulation

*Associated with Universiti Kuala Lumpur*

The Teras Jernang Wastewater Treatment Plant Automation Simulation project was developed as part of my PLC subject in university. This project involved the simulation and control of various processes within a wastewater treatment plant using Omron PLCs (CQM1H CPU51 and CS1G(H) CPU42/44). As the lead programmer, I created control algorithms to manage the operations of the plant, including the inlet works, forward feed pumping station, storm tank, aeration tank, clarifier and RAS/WAS pumps, sludge holding tanks, tertiary filter feed sump, tertiary treatment filter, and the final effluent pumping station.

**Skills**: Omron PLC Programming 路 PLC Programming 路 Industrial Networking 路 Wastewater Treatment 路 SCADA
